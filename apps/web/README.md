# Web

This is the web application for the InfocusApp application. It is built with Next.js and uses Tailwind CSS for styling.

## Getting Started

To get started with the web application, you first need to install the dependencies:

```bash
pnpm install
```

Then, you can run the development server:

```bash
pnpm dev
```

This will start the server on `http://localhost:3000`.

## Building

To build the application for production, you can run the following command:

```bash
pnpm build
```
